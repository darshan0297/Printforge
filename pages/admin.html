🌐 Get a Free @printforge.lk Email

Two free ways to get a professional email address on your domain. Pick one.

☁️
Cloudflare Email Routing
Free
Forwards info@printforge.lk to your Gmail. You reply from Gmail, customers see your domain address.

Steps:
1. Move DNS to Cloudflare (free)
2. Go to Email → Email Routing
3. Add rule: info@printforge.lk → darshan0297@gmail.com
4. In Gmail Settings → "Send mail as" → add info@printforge.lk via SMTP
Open Cloudflare →
📮
Zoho Mail
Free · 5 users
Full mailbox at info@printforge.lk with webmail, mobile app, and 5GB storage.

Steps:
1. Sign up at zoho.com/mail (free plan)
2. Add domain: printforge.lk
3. Add MX records to domains.lk DNS panel
4. Create info@printforge.lk mailbox
Open Zoho Mail →
DNS records to add in domains.lk (for Zoho Mail):
MX    @    mx.zoho.com    Priority 10
MX    @    mx2.zoho.com   Priority 20
TXT   @    v=spf1 include:zoho.com ~all
⚡ Transactional Emails (Order Confirmations)

To auto-send order confirmation emails to customers, add Resend to your Supabase project. 3,000 free emails/month.

1. Sign up at resend.com (free)
2. Add & verify printforge.lk domain
3. Copy your Resend API key
4. Go to Supabase → Edge Functions → Deploy send-order-email function
5. Set RESEND_API_KEY in Supabase secrets

Range multiplier sets the high estimate — e.g. 1.6 means "LKR 1,000 — LKR 1,600"

📐 Material Base Rates (LKR per small unit)

Base rate = cost for a small (10×10cm) cut-only job. Operations and size multiply from here.

📏 Size Multipliers

Each size tier multiplies the base rate. Small (10×10cm) = 1.0 is the reference.

⚙️ Operation Multipliers

Cut = 1.0 reference. Operations that take more time get a higher multiplier.
