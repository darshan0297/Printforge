📋 File prep tips
✅Export as binary STL from Blender: File → Export → STL. Keep scale at 1.0 and apply modifiers.
✅Make sure the model is watertight (manifold) — no holes, no inverted normals.
✅Max build volume: 220 × 220 × 300 mm. Larger models can be split and glued.
✅Design minimum wall thickness of 1.2mm (3× nozzle width) for reliable walls.
⚠️Overhangs over 45° need supports unless you've designed in chamfers.
💡Not sure? Send us a reference image and dimensions — we can model simple parts for you.
